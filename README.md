# Vane Academy (Vane Matrix Academy) GitHub Pages Website

Upload all files in this folder to your GitHub Pages repository.

## Included pages
- index.html
- about.html
- faq.html
- insights.html
- contact.html
- article-1.html to article-7.html
- style.css
- script.js
- robots.txt
- sitemap.xml
- organization-schema.json

## Required images already in your GitHub repository
The HTML references your uploaded image names directly:
Logomark.png, logo.png, logo-red.jpg, logo-white.jpg, building.png, building-2.png, front-desk.png, office-1.png, office-2.png, poster-1.jpg to poster-7.jpg, and social media icons inside the `social media` folder.

## GitHub Pages
After uploading, open Settings > Pages and publish from the main branch root.
